25 March 2013

I decided put my basic dotfiles and vim setup on github and create a way to sync them between my computers. I've yet to switch to vim on my personal computer and figured this would be a good time to create a little setup.  The setup was mostly inspired by Ryan Bates. In the midst of it I also installed z shell while I was looking at his. For my personal alias' I was almost going to them in there but decided to keep those separate per computer in my ~/.bash_profile and source it in my .zshrc file.

You can checkout my setup on github.
